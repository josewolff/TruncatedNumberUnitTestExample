Language and tools used:
    Java    - Main Language.
    Maven   - Manage dependencies, compile and excute the code.
    TestNG  - To make the unit tests.

To execute the program you need to use the following command:

    Command: compile exec:java -Dexec.mainClass=exercise.Main -Dexec.args=numberYouWantToUse.

To execute the tests use the following command:

    Command: mvn test

TestNG
    - Used dataProviders that helps to execute the test with different parameters.
    - Boundary Values - Use the limits to create the automated tests.
